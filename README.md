An experimental, story-driven photography portfolio built with GSAP and ScrollTrigger. This project explores how motion, typography, and layout can work together to create a cinematic, immersive web experience rather than a traditional image gallery.

## Overview

Instead of a static portfolio, this project is designed as a guided visual narrative. Each scroll reveals a new layer of the story through animation, spacing, and controlled transitions.

The focus is on:
- Cinematic scroll-based storytelling  
- High-end motion design using GSAP  
- Minimal, intentional UI structure  
- Strong visual hierarchy and pacing  
- Awwwards-style interaction patterns  

## Tech Stack

- HTML5  
- CSS3  
- JavaScript (Vanilla)  
- GSAP  
- ScrollTrigger

## Features

- Scroll-driven narrative structure  
- Pinned sections with animated transitions  
- Parallax and layered motion effects  
- Interactive portfolio grid with hover states  
- Smooth, cinematic user experience  

## Concept Goal

This project is not meant to be a traditional portfolio. It’s an exploration of how a photographer’s work can be presented as an experience—where motion and storytelling are as important as the imagery itself.
